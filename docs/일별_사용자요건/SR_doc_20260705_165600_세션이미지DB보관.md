## 처리 내역

- 이미지 검색 성공 직후 `POST /api/search-images`로 서버 DB(Vercel Blob)에 썸네일 저장
- 세션 쿠키(`sc-session`, 30일)로 이미지·이력 연결
- localStorage에는 `imageId`·`imageUrl`만 저장 (용량 절약), 세션 만료·캐시 삭제 후에도 DB에서 복원
- 앱 로드 시 `GET /api/search-images`로 DB 이미지 이력 병합
- 개발 환경: `.data/search-images/` 파일 폴백
- 프로덕션: `BLOB_READ_WRITE_TOKEN` 필요 (미설정 시 로컬 썸네일 폴백)
